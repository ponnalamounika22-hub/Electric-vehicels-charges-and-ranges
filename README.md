# Electric-vehicels-charges-and-ranges

This project presents an interactive dashboard for analyzing electric vehicle (EV) charging infrastructure and vehicle performance in India. It enables users to compare EV brands based on driving range, charging speed, efficiency, top speed, price, body style, and powertrain. The dashboard provides meaningful visualizations to support data-driven insights into the growing EV ecosystem.

📁 1. Brainstorming & Ideation

Problem Statement:

The increasing adoption of electric vehicles has generated a large amount of data related to battery capacity, charging speed, driving range, vehicle efficiency, and pricing. Analyzing this information manually is difficult. This project aims to develop an interactive visualization dashboard that helps users understand EV performance and charging characteristics through meaningful charts and dashboards.

* Brainstorming

During brainstorming, several ideas were discussed:

•	EV charging station analysis 
•	Battery range comparison 
•	Vehicle efficiency analysis 
•	Brand-wise EV comparison 
•	Price analysis 
•	Dashboard development using Tableau 

The final idea selected was Visualization Tool for Electric Vehicle Charge and Range Analysis.

Literature Survey

Various research papers, EV manufacturer websites, and government reports were studied to understand electric vehicle technology, charging infrastructure, and battery performance. Existing visualization tools were also analyzed to identify useful dashboard features.

Objectives

•	Analyze EV datasets efficiently. 

•	Visualize charging speed and driving range. 

•	Compare different EV brands. 

•	Study vehicle efficiency. 

•	Build an interactive Tableau dashboard.
